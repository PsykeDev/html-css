Usando links externos

É um link que vai dá acesso para outros sites que não seja o meu. EX: Repositorio GitHub do gustavoguanabara

Como abrir links externos em outra aba?

Basta usar dois comando dentro do comando <a href="">

R: target="_blank" e rel="external"

<a href="http://gustavoguanabara.github.io/" target="_blank" rel="external </a>
